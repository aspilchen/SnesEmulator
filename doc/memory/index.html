Module snes::memory

source ·
Expand description

Controls memory IO.

The SNES uses memory maps to access RAM/ROM etc. A lot of the mapping is determined by the cartridge, and there are a number of different mapping structures depending on which cartridge is used. This will eventually serve as an interface to the memory map, and therefore all memory IO should go through here.
